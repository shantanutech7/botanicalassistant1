# 🌿 Smart Botanical Assistant

An AI-powered plant identification and health analysis system that combines computer vision (YOLOv8) with Claude AI to provide intelligent botanical insights.

---

## 🚀 Features

- 🔍 **Plant Disease Detection** — YOLOv8-based real-time image analysis
- 🤖 **Claude AI Integration** — Deep botanical insights and care recommendations
- 🌱 **Health Status Analysis** — Instant plant health diagnosis
- 💬 **Hinglish Support** — Recommendations in Hindi + English
- 📊 **Confidence Scoring** — Detection accuracy percentage
- 🖼️ **Image Upload** — Simple drag and drop interface

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Tailwind CSS |
| Backend | Python, Flask |
| AI/ML | YOLOv8, Claude AI (Anthropic) |
| Computer Vision | OpenCV, PIL |

---

## ⚙️ Installation & Setup

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd app
npm install
npm run dev
```

## 📸 How It Works

1. Upload a plant image
2. YOLOv8 detects disease/health status
3. Claude AI generates detailed care recommendations
4. Get instant results with confidence score
